Keen — First Launch (macOS)

1) Install
- Open the Keen.dmg
- Drag Keen.app into the Applications folder

2) First open (if macOS blocks it)
- Open System Settings
- Go to Privacy & Security
- Scroll to the bottom and click “Open Anyway” for Keen

3) Advanced (Terminal)
If you know what you’re doing, you can remove the quarantine flag:

xattr -dr com.apple.quarantine /Applications/Keen.app
